# ⚙️ MechLab

**MechLab** is a web-based interactive tool designed to help students, engineers, and enthusiasts explore and visualize fundamental topics in mechanical engineering, including stress analysis, kinematics, and machine elements.

## Available SPAs

- [Stress transformation](https://jorgedelossantos.github.io/mech-lab/stress-transformation)
- [Stepped bar under axial load](https://jorgedelossantos.github.io/mech-lab/stepped-bar-axial-load)
